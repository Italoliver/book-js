## [Cap00 - Introdução](Capítulos/introducao.md)

## [Cap01 - O que é JavaScript e por que ele importa](Capítulos/cap01.md)
- História da linguagem
- Onde o JavaScript é usado
- Como ele funciona no navegador
- O papel do JS no desenvolvimento moderno
#### Este capítulo apresenta o JavaScript de forma simples e motivadora. O leitor entenderá o que é a linguagem, como ela surgiu, por que se tornou tão popular e onde é usada hoje — navegadores, servidores, aplicativos, automações e muito mais. A ideia é mostrar o impacto real do JavaScript no mundo moderno e despertar curiosidade.

## [Cap02 - Preparando o ambiente](Capítulos/cap02.md)
- Instalando um editor de código
- Console do navegador
- Criando seu primeiro arquivo `.js`
- Como conectar JavaScript ao HTML
#### Aqui o leitor monta seu primeiro ambiente de desenvolvimento. Você explica como instalar um editor de código, usar o console do navegador, criar arquivos HTML e JS e conectar os dois. O objetivo é deixar tudo pronto para começar a programar sem complicações.

## [Cap03 - Variáveis e tipos de dados](.md)
- `let`, `const` e `var`
- Tipos primitivos
- Boas práticas de nomeação
- Conversões de tipo
#### O primeiro contato com código de verdade. O leitor aprende o que são variáveis, como declarar com let e const, quais são os tipos primitivos (string, number, boolean, null, undefined, symbol, bigint) e como eles funcionam. Também aborda boas práticas de nomeação e conversões de tipo.

## [Cap04 - Operadores e expressões](.md)
- Operadores matemáticos
- Operadores lógicos
- Comparações
- Precedência e agrupamento
#### Este capítulo ensina como o JavaScript realiza cálculos e decisões. Você apresenta operadores matemáticos, lógicos, de comparação e atribuição. Também explica precedência, agrupamento e como montar expressões que o computador entende.

## [Cap05 - Estruturas de controle](.md)
- `if`, `else`, `else if`
- `switch`
- Loops: `for`, `while`, `do...while`
- Quando usar cada um
#### Agora o leitor aprende a controlar o fluxo do programa. Você explica `if`, `else`, `switch`, loops (`for`, `while`, `do...while`) e quando usar cada um. O foco é mostrar como o código toma decisões e repete ações.

## [Cap06 - Funções: o coração do JavaScript](.md)
- Declaração e execução
- Parâmetros e retorno
- Arrow functions
- Escopo e hoisting
#### Um dos capítulos mais importantes. Aqui o leitor aprende a criar funções, passar parâmetros, retornar valores, usar arrow functions e entender escopo e hoisting. O objetivo é mostrar como funções organizam o código e evitam repetição.

## [Cap07 - Arrays e objetos](.md)
- Criando e manipulando arrays
- Métodos essenciais (`map`, `filter`, `reduce`)
- Objetos e propriedades
- Acessando e modificando dados
#### O leitor descobre como armazenar e organizar dados. Você explica como criar arrays, acessar elementos, usar métodos essenciais (`map`, `filter`, `reduce`) e trabalhar com objetos, propriedades e estruturas mais complexas. Este capítulo prepara o terreno para manipulação de dados real.

## [Cap08 - Manipulação do DOM](.md)
- Selecionando elementos
- Alterando conteúdo e estilos
- Criando e removendo elementos
- Eventos e interatividade
#### Agora o JavaScript ganha vida na página. O leitor aprende a selecionar elementos, alterar textos e estilos, criar e remover elementos e reagir a eventos como cliques e teclas. É aqui que a programação começa a ficar visual e divertida.

## [Cap09 - Assíncronismo: entendendo o tempo no JavaScript](.md)
- Callbacks
- Promises
- `async` e `await`
- Como lidar com requisições externas
#### Um dos temas mais importantes da linguagem. Você explica o que é assíncronismo, como funcionam callbacks, promises e `async/await`, e como lidar com requisições externas. O objetivo é desmistificar o assunto e mostrar que não é tão complicado quanto parece.

## [Cap10 - Boas práticas e próximos passos](.md)
- Organização de código
- Debugging
- Modularização
- Caminhos para avançar: APIs, frameworks, backend com Node.js
#### O capítulo final reúne orientações para escrever código limpo, organizado e fácil de manter. Você fala sobre debugging, modularização, padrões de escrita e caminhos para continuar evoluindo: APIs, frameworks, Node.js, bibliotecas e projetos práticos.
